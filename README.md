## 0x3 脚本使用说明

### 3.1 北京顺义妇幼保健院

文件目录为：SYFYBJY

- 捡漏脚本：JL.py
- 抢票脚本：QP.py

只需要抓包顺义妇幼保健院提取出每个人的Cookie，填写到Cookie参数即可，北京顺义妇幼保健院的机房是北京阿里云的，挂脚本建议选择北京阿里云的机房

### 3.2 北京今康中医院

文件目录为：BJJKZ

- 抢票脚本：QP.py

只需要抓包北京今康中医院的查询之前的预约界面，提取出每个人的Cookie与URL，然后修改要POST包的数据和URL链接即可
